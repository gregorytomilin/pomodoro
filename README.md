# Pomodoro

Hello and welcome to pomodoro time manager

To start develope please use command below

```
npm run dev
```
To build application please use command below

```
npm run build
```

## Stack

- **TypeScript**
- **React JS**
- **Effectror**
- **React Router**
- **Vite**
- **ESLint**
- **Prettier**
